วิธีใช้
  1. หัวข้อจะถูกล็อก ตามเมนูที่คุณกดใน Rich Menu (เพื่อไม่ให้สลับผิดหัวข้อ)
  2. เลือกคลิปจาก “รายการคลิป” แล้วกด ดูคลิป
  3. กด ดูต่อ เพื่อกลับไปคลิปล่าสุดของหัวข้อนี้
  4. ในหน้าวิดีโอ กด ทำเครื่องหมายดูแล้ว เพื่อให้ระบบจำในเครื่องนี้ (ถ้าเครื่องอนุญาต)
  5. ถ้าต้องการเปลี่ยนหัวข้อ ให้ปิดหน้านี้แล้วกลับไปกด Rich Menu ใหม่
