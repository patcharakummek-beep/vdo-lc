วิธีใช้
  1. เลือกหัวข้อ A/B/C จากดรอปดาวน์
  2. กด “ดูคลิป” เพื่อเปิดวิดีโอ (เล่นจาก Google Drive)
  3. กด “ดูแล้ว” เพื่อให้ระบบจำในเครื่อง (ถ้าเครื่องอนุญาต storage)
  4. กด “แชร์” เพื่อส่งให้ญาติ/ผู้ดูแล
